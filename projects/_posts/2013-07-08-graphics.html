The highest-numbered undergrad CS elective offered at UVa is CS 4810, Introduction to Computer Graphics. Its transparent grading policy, rewarding results, and unforgivingly mathematical nature made it a fun challenge and certainly the most worthwhile class I've ever taken.

Sixty percent of the final grade was determined by three fairly intense C++ programming assignments, which were each accompanied by an extra-credit art contest that required entries to be generated using the programs themselves. I won all three contests; my submissions are below (click to enlarge/animate).

Morphing and crossfading in the first two entries were achieved using hardcoded sets of control lines and the Beier-Neely algorithm, famously featured in the music video for Michael Jackson's Black or White.

Of course, I can't take all the credit. The instructor did provide students with some precooked resources: class method signatures, matrix arithmetic functions*, and a basic I/O framework. Our job was to make sense of how the given classes were supposed to interact with each other (the "documentation" wasn't too helpful), write their methods in accordance with the Phong reflection model, construct scene data files by hand, and attempt to debug visual defects using little more than std::cout and patience.

Detailed reports:

  • Assignment 1: Bitmap image processing
  • Assignment 2: Raytracer
  • Assignment 3: OpenGL

*I actually found a rather egregious bug in the provided routine for ray transformation in homogeneous coordinates that caused all normals on one side of an arbitrary plane to be reversed. I'm guessing that such a showstopper remained unfixed for so long only because we were explicitly told not to modify the matrix library (although to his credit, my instructor probably wasn't the one who wrote the code he gave us).
