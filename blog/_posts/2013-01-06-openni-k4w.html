Hopefully this will save some poor soul from many more hours of frustration. Please note that this is for Kinect for Windows, not the regular Xbox Kinect.

  1. Remove whatever versions of OpenNI and SensorKinect you may already have on your computer.
  2. Install the patched version of OpenNI 1.5.4 from PCL.
  3. Install the latest version of SensorKinect from avin2.
  4. Install the latest unstable version of OpenNI 2.

This method worked today on my 64-bit Windows 7 laptop (official Kinect SDK not installed). No guarantees that it'll work for much longer...
